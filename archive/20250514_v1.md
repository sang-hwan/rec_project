  - 팔로우 계정 ID 목록 정리 (엑셀/CSV)

- **데이터 저장 환경 설정**
  - PostgreSQL 설치 및 테이블 생성
  - 데이터 저장 Python 스크립트 작성

- ChatGPT를 통한 분석 진행 (약 20분)

### ✅ 5단계: 정량 분석 진행 (매일 또는 주 2~3회)
